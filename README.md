# Customer Records Tracker

## Project Description
The **Customer Records Tracker** is a lightweight, backend-focused JavaScript application designed to efficiently manage and manipulate small-business customer data. Utilizing advanced compound data structures—specifically arrays of complex objects—this system handles dynamic customer profiles, updates multi-layered purchase histories, and generates formatted data reports. 

This project demonstrates proficiency in foundational JavaScript data manipulation, array methods, and data iteration protocols required for modern business application development.

## Key Features
*   **Dynamic Inventory/Client Management:** Leverages mutator methods like `.push()` and `.shift()` to handle real-time data ingestion and queue management.
*   **Multi-layered Object Architecture:** Structures client profiles with nested attributes, isolating complex data arrays (like individual purchase histories) within standalone parent objects.
*   **Automated Reporting:** Iterates through high-level arrays using `.forEach()` logic and dynamically renders localized template literals tracking core customer engagement metrics.

## Technologies Used
*   **Language:** JavaScript (ES6+)
*   **Environment:** Node.js / Browser Console
*   **Version Control:** Git & GitHub
